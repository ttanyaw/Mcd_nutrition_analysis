McDonald’s Menu Nutrition Analysis (Power BI)

This project analyzes the nutritional composition of McDonald’s menu items using Power BI, with a focus on how portion size impacts calories, fat, sugar, and protein.
The dashboard is designed to provide clear, average-based nutritional insights per menu item, avoiding misleading totals.

Objectives

Analyze average nutritional values across menu categories. 

Compare Small, Medium, and Large portion sizes.

Identify high-calorie and high-fat items.

Present insights in an interactive Power BI dashboard.
